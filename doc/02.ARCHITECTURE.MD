## 1. Обзор архитектуры

### Политика входного формата (важно)

- Входной контейнер `.mkv` поддерживается **напрямую**.
- Предварительная конвертация `.mkv` → `.mp4` **не требуется**.
- Обработка делается через `ffmpeg`/`ffprobe` из исходного файла.
- Для DEMO вырезается 1 минута из исходника без полной перекодировки файла.
- Поддерживаемые входные расширения в MVP: `.mkv`, `.mp4`.
- Опциональная конвертация будет добавлена только как fallback (не как обязательный шаг).

### 3.3 `whisper_handler.py`

**Назначение**: Интеграция faster-whisper для транскрибации

**Зависимости**: `faster-whisper`, `pathlib`

**Интерфейс**:
```python
class WhisperHandler:
    def __init__(self, model_name: str = "medium", device: str = "auto"):
        from faster_whisper import WhisperModel
        self.model = WhisperModel(model_name, device=device)
    
    def transcribe(self, audio_file: Path) -> List[Segment]:
        """Транскрибировать аудиофайл."""
        segments, info = self.model.transcribe(
            str(audio_file),
            language="ru",  # Russian + Ukrainian
            beam_size=5,    # Balance between speed and quality
        )
        
        return [
            Segment(
                start=seg.start,
                end=seg.end,
                text=seg.text
            )
            for seg in segments
            if len(seg.text.strip()) >= 2  # MIN_TEXT_LEN
        ]

@dataclass
class Segment:
    start: float      # Начало в секундах
    end: float        # Конец в секундах
    text: str         # Текст сегмента
```

**Разница с OpenAI API**:
```python
# Old (OpenAI API)
response = client.audio.transcriptions.create(file=f, model="whisper-1")
segments = response.segments

# New (faster-whisper)
segments, info = model.transcribe(audio_file)
```

### 3.4 `dialog_builder.py`

**Назначение**: Сборка диалога из сегментов двух спикеров

**Входные данные**: 
- Сегменты S1 (левый канал)
- Сегменты S2 (правый канал)
- Глобальный offset для фрагмента

**Выходные данные**: Отсортированный диалог с чередованием спикеров

**Интерфейс**:
```python
@dataclass
class DialogLine:
    time: float       # Начало фразы
    end: float        # Конец фразы
    speaker: str      # S1 или S2
    text: str         # Текст фразы

class DialogBuilder:
    def merge(
        self,
        segments_s1: List[Segment],
        segments_s2: List[Segment],
        global_offset: float = 0.0,
    ) -> List[DialogLine]:
        """Объединить два потока спикеров в один диалог."""
        dialog = []
        
        # Добавить S1
        for seg in segments_s1:
            dialog.append(DialogLine(
                time=seg.start + global_offset,
                end=seg.end + global_offset,
                speaker="S1",
                text=seg.text
            ))
        
        # Добавить S2
        for seg in segments_s2:
            dialog.append(DialogLine(
                time=seg.start + global_offset,
                end=seg.end + global_offset,
                speaker="S2",
                text=seg.text
            ))
        
        # Отсортировать по времени
        dialog.sort(key=lambda x: x.time)
        return dialog
    
    def save_txt(self, dialog: List[DialogLine], output_file: Path) -> None:
        """Сохранить диалог в текстовый формат."""
        with output_file.open("w", encoding="utf-8") as f:
            for line in dialog:
                f.write(f"[{self._format_time(line.time)}] "
                       f"{line.speaker}: {line.text}\n")
    
    def save_srt(self, dialog: List[DialogLine], output_file: Path) -> None:
        """Сохранить диалог в SRT формат."""
        with output_file.open("w", encoding="utf-8") as f:
            for idx, line in enumerate(dialog, 1):
                f.write(f"{idx}\n")
                f.write(f"{self._format_time_srt(line.time)} --> "
                       f"{self._format_time_srt(line.end)}\n")
                f.write(f"{line.speaker}: {line.text}\n\n")
    
    @staticmethod
    def _format_time(seconds: float) -> str:
        """Формат: HH:MM:SS.mmm"""
        h = int(seconds // 3600)
        m = int((seconds % 3600) // 60)
        s = seconds % 60
        return f"{h:02}:{m:02}:{s:06.3f}"
    
    @staticmethod
    def _format_time_srt(seconds: float) -> str:
        """Формат SRT: HH:MM:SS,mmm"""
        return DialogBuilder._format_time(seconds).replace(".", ",")
```

### 3.5 `logger.py`

**Назначение**: Логирование обработки в JSON формате

**Выходные данные**: `processing.log.json` с полной информацией

**Интерфейс**:
```python
class ProcessingLogger:
    def __init__(self, log_file: Path):
        self.log_file = log_file
        self.start_time = datetime.now(timezone.utc)
        self.stages = []
    
    def log_stage(
        self,
        name: str,
        status: str,  # running, completed, failed
        duration: float = None,
        details: dict = None,
    ) -> None:
        """Залогировать этап обработки."""
        self.stages.append({
            "name": name,
            "status": status,
            "duration_sec": duration,
            "details": details or {},
            "timestamp": datetime.now(timezone.utc).isoformat(),
        })
    
    def save(self) -> None:
        """Сохранить лог в JSON."""
        log_data = {
            "start_time": self.start_time.isoformat(),
            "end_time": datetime.now(timezone.utc).isoformat(),
            "total_duration_sec": (
                datetime.now(timezone.utc) - self.start_time
            ).total_seconds(),
            "stages": self.stages,
            "status": "completed" if all(s["status"] != "failed" for s in self.stages) else "failed",
        }
        
        with self.log_file.open("w", encoding="utf-8") as f:
            json.dump(log_data, f, ensure_ascii=False, indent=2)
```

## 4. Главный скрипт (main.py)

### 4.1 Структура main.py

```python
#!/usr/bin/env python3
"""
Main entry point for v2t-decoder video transcription system.
"""

import sys
from pathlib import Path
from datetime import datetime
import argparse

# Import modules
from src.config_loader import load_config
from src.ffmpeg_handler import FFmpegHandler
from src.whisper_handler import WhisperHandler
from src.dialog_builder import DialogBuilder
from src.logger import ProcessingLogger
from src.errors import *

def main():
    # Этап 1: Parse CLI arguments
    parser = argparse.ArgumentParser(description="v2t-decoder: Video transcription system")
    parser.add_argument("--config", type=Path, default=Path("config.yaml"),
                       help="Path to config file")
    args = parser.parse_args()
    
    # Этап 2: Load config
    try:
        config = load_config(args.config)
    except ConfigError as e:
        print(f"CONFIG ERROR: {e}", file=sys.stderr)
        sys.exit(1)
    
    # Этап 3: Validate input
    if not config.video_file.exists():
        print(f"ERROR: Video file not found: {config.video_file}", file=sys.stderr)
        sys.exit(1)
    
    # Этап 4: Create output directory
    config.output_dir.mkdir(parents=True, exist_ok=True)
    
    # Этап 5: Initialize logger
    logger = ProcessingLogger(config.output_dir / "processing.log.json")
    logger.log_stage("init", "running")
    
    # Этап 6: Initialize handlers
    ffmpeg = FFmpegHandler()
    whisper = WhisperHandler(model_name=config.whisper_model, device=config.device)
    dialog_builder = DialogBuilder()
    
    try:
        # Этап 7: Get video info
        logger.log_stage("video_info", "running")
        duration = ffmpeg.get_duration(config.video_file)
        audio_streams = ffmpeg.get_audio_streams(config.video_file)
        
        if not audio_streams:
            raise NoAudioError(f"No audio streams found in {config.video_file}")
        
        logger.log_stage("video_info", "completed", details={
            "duration_sec": duration,
            "audio_streams": audio_streams,
        })
        
        # Этап 8: Determine processing duration
        if config.mode == "demo":
            process_duration = config.demo_duration * 60  # minutes to seconds
        else:
            process_duration = min(duration, config.max_duration * 60)
        
        # Этап 9: Extract audio
        logger.log_stage("extract_audio", "running")
        temp_dir = config.output_dir / "extracted_audio"
        temp_dir.mkdir(exist_ok=True)
        
        s1_mp3 = temp_dir / "s1_fl.mp3"
        s2_mp3 = temp_dir / "s2_fr.mp3"
        
        ffmpeg.extract_channel(config.video_file, audio_streams[0], "FL", s1_mp3)
        
        # Обработать S2: если есть второй канал, использовать его; иначе дублировать S1
        if len(audio_streams) > 1:
            ffmpeg.extract_channel(config.video_file, audio_streams[1], "FR", s2_mp3)
        else:
            # Моно видео - дублировать S1
            import shutil
            shutil.copy(s1_mp3, s2_mp3)
        
        logger.log_stage("extract_audio", "completed", details={
            "s1_file": str(s1_mp3),
            "s2_file": str(s2_mp3),
        })
        
        # Этап 10: Transcribe
        logger.log_stage("transcribe", "running")
        
        segments_s1 = whisper.transcribe(s1_mp3)
        logger.log_stage("transcribe_s1", "completed", details={
            "segments": len(segments_s1),
        })
        
        segments_s2 = whisper.transcribe(s2_mp3)
        logger.log_stage("transcribe_s2", "completed", details={
            "segments": len(segments_s2),
        })
        
        # Этап 11: Merge dialog
        logger.log_stage("merge_dialog", "running")
        dialog = dialog_builder.merge(segments_s1, segments_s2, global_offset=0.0)
        logger.log_stage("merge_dialog", "completed", details={
            "total_lines": len(dialog),
        })
        
        # Этап 12: Save outputs
        logger.log_stage("save_output", "running")
        
        txt_file = config.output_dir / "dialog_timeline.txt"
        srt_file = config.output_dir / "dialog_timeline.srt"
        
        dialog_builder.save_txt(dialog, txt_file)
        dialog_builder.save_srt(dialog, srt_file)
        
        logger.log_stage("save_output", "completed", details={
            "txt_file": str(txt_file),
            "srt_file": str(srt_file),
        })
        
        # Этап 13: Save logger
        logger.save()
        
        print(f"✅ COMPLETED")
        print(f"  TXT: {txt_file}")
        print(f"  SRT: {srt_file}")
        print(f"  LOG: {config.output_dir / 'processing.log.json'}")
        
        return 0
        
    except NoAudioError as e:
        print(f"❌ VIDEO ERROR: {e}", file=sys.stderr)
        logger.log_stage("error", "failed", details={"error": str(e)})
        logger.save()
        sys.exit(1)
    
    except Exception as e:
        print(f"❌ ERROR: {e}", file=sys.stderr)
        logger.log_stage("error", "failed", details={"error": str(e)})
        logger.save()
        sys.exit(1)

if __name__ == "__main__":
    sys.exit(main())
```

## 8. Workflow (DEMO mode)

### 8.1 Пошаговый процесс (1 минута видео)

```
1. USER: python main.py --config config.yaml
         (config.yaml содержит путь к видео)
         
2. LOAD CONFIG
   → Прочитать YAML
   → Валидировать параметры
   → Создать директорию output/
   
3. VALIDATE VIDEO
   → Проверить что видео существует
   → Получить длительность (ffprobe)
   → Получить аудиопотоки (ffprobe)
   → Проверить: есть ли аудио? ✅
   
4. INIT HANDLERS
   → FFmpegHandler()
   → WhisperHandler(model="medium", device="auto")
   → DialogBuilder()
   → ProcessingLogger()
   
5. EXTRACT AUDIO (DEMO: первая 1 минута)
   [00:00-01:00] → ffmpeg -i video -map 0:a:0 -af pan=mono|c0=FL s1.mp3
   [00:00-01:00] → ffmpeg -i video -map 0:a:1 -af pan=mono|c0=FR s2.mp3
   
6. TRANSCRIBE S1
   whisper_model.transcribe(s1.mp3) →
   [
     {start: 0.0, end: 1.5, text: "With your permission."},
     {start: 19.18, end: 44.18, text: "Привет, Зайчик, привет, Марусечка..."},
     ...
   ]
   
7. TRANSCRIBE S2
   whisper_model.transcribe(s2.mp3) →
   [
     {start: 30.0, end: 33.0, text: "Молодец, ты и бежал."},
     ...
   ]
   
8. MERGE DIALOG
   Объединить S1 и S2 сегменты
   Отсортировать по времени
   →
   [
     {time: 0.0, end: 1.5, speaker: S2, text: "With your permission."},
     {time: 19.18, end: 44.18, speaker: S2, text: "Привет, Зайчик..."},
     {time: 30.0, end: 33.0, speaker: S1, text: "Молодец, ты и бежал."},
     ...
   ]
   
9. SAVE OUTPUTS
   → output/dialog_timeline.txt
   → output/dialog_timeline.srt
   → output/processing.log.json
   
10. EXIT
    ✅ COMPLETED
    TXT: ./output/dialog_timeline.txt
    SRT: ./output/dialog_timeline.srt
    LOG: ./output/processing.log.json
```

## 11. Граничные случаи (Edge Cases)

| Случай | Входные данные | Ожидаемый результат | Статус |
|--------|-----------------|-------------------|----|
| Видео без аудио | `video_no_audio.mp4` | ❌ FATAL: "No audio streams" | Обработано |
| Моно видео (1 канал) | `video_mono.mp4` | ⚠️ WARNING + S2=S1 дублирована | Обработано |
| Видео 20 часов | `video_20h.mkv` | ⚠️ WARNING + обработано 2 часа | Обработано |
| Неподдерживаемый формат | `file.avi` | ❌ ERROR | Обработано |
| Видео с 4 каналами | `video_4ch.mkv` | ℹ️ INFO + используются первые 2 | Обработано |
