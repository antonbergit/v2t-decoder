## 🎯 Рекомендуемые решения (на основе анализа)

### 1. Whisper модель: **`faster-whisper` (medium)**
- ✅ Работает локально без API
- ✅ Скорость: 5-10 сек на минуту аудио
- ✅ Качество: близко к `whisper-1` API
- ✅ Память: 4-8 GB (приемлемо для ProxmoxVE)
- 📦 Установка: `pip install faster-whisper`

### 2. Архитектура: **Прямая замена + wrapper функция**
```python
def transcribe_whisper(model, audio_file: Path, **kwargs):
    """Унифицированный интерфейс для обоих способов."""
    # Работает с faster_whisper.WhisperModel
    # Возвращает тот же формат, что и OpenAI API
```

### 3. CLI: **`argparse` + опциональный конфиг YAML**
```bash
# Быстрый запуск
python main.py input.mkv --demo

# Производство
python main.py input.mkv --production --fragment-size 10

# С конфигом
python main.py --config config.yaml
```

### 4. Выходные форматы: **`.txt` + `.srt` + `.json`**
- `.json` для программной обработки результатов
- `.txt` и `.srt` для человека и плеера

### 5. Контейнер: **Docker**
- Портабельность (работает в ProxmoxVE, локально, облаке)
- Управление зависимостями
- Воспроизводимость окружения
