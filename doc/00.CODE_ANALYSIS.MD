### 2.1 `whisper_probe.py` — Диагностика (69 строк)

**Цель**: быстрая проверка интеграции с Whisper API

**Входные данные**:
- Первый найденный видеофайл (`.mp4` / `.mkv`)
- API ключ из `api.key`

**Алгоритм**:
1. Открыть видеофайл
2. Извлечь первую аудиодорожку в моно (128k MP3)
3. Отправить на `client.audio.transcriptions.create()` с параметрами:
   - `model="whisper-1"`
   - `response_format="verbose_json"`
   - `prompt="Court hearing recording. Languages: Russian and Ukrainian. Multiple speakers."`
4. Сохранить RAW JSON ответ в `whisper_raw_response.json`

**Статус**: ✅ Работает, но специализирован только на диагностике

**Код цикла**:
```python
with mp3.open("rb") as f:
    response = client.audio.transcriptions.create(
        file=f,
        model="whisper-1",
        response_format="verbose_json",
        prompt=PROMPT,
    )
response_dict = response.model_dump()
```

**Выход**:
```json
{
  "text": "...",

      "id": 0,

      "avg_logprob": -0.2,
```

```python
for stream in video.audio_streams:
    # 1. Оригинальный поток (WAV)
    ffmpeg -i video -map 0:{stream} output_orig.wav
    
    # 2. Моно микс (downmix оба канала)
    ffmpeg -i video -map 0:{stream} -ac 1 output_mono.wav

    ffmpeg -i video -map 0:{stream} -af "pan=mono|c0=FL" output_left.wav
```

### 2.3 `transcribe_from_extracted_audio.py` — Альтернативный pipeline (164 строки)

**Цель**: транскрибация уже извлечённых аудиофайлов

**Входные данные**:
- MP3 файлы в директории `extracted_audio/`
- Конвенция: `{stem}_S1_FL.mp3` и `{stem}_S2_FR.mp3`

**Ключевые параметры**:
```python
OFFSET_PER_PAIR_SEC = 600        # 10 минут смещение между парами файлов
SPEAKERS = {"S1": "FL", "S2": "FR"}
MAX_NO_SPEECH_PROB = 0.6         # Фильтр паузы (вероятность > 60% пропускается)
MIN_TEXT_LEN = 2                 # Минимум 2 символа в сегменте
```

**Алгоритм**:
```python
1. Загрузить все MP3 файлы из extracted_audio/
2. Сгруппировать по {stem} (видеофайлу)
3. Для каждой группы:
   3.1 Для каждого MP3:
       - Определить спикера (S1/S2)
       - Отправить на Whisper
       - Отфильтровать сегменты:
         * Текст >= 2 символов
         * no_speech_prob <= 0.6
   3.2 Добавить к диалогу с глобальным offset:
       - time = segment.start + global_offset
       - end = segment.end + global_offset
   3.3 Увеличить global_offset на 600 сек
4. Отсортировать диалог по времени
5. Сохранить в .txt и .srt форматы
```

**Выходные файлы**:
- `{stem}_dialog.txt`
- `{stem}_dialog.srt`

**Статус**: ✅ Работает полностью, но требует подготовки файлов

### 2.4 `transcribe_calls.py` ⭐ **ОСНОВНОЙ** (199 строк)

**Цель**: полный автоматический pipeline "видео → диалог"

**Входные данные**:
- Видеофайлы (`.mp4` / `.mkv`)
- API ключ из `api.key`

**Ключевые параметры**:
```python
SUPPORTED_VIDEO_EXT = {".mp4"}
STREAM_INDEX = 1                 # Индекс аудиопотока в видео
SPEAKERS = {"S1": "FL", "S2": "FR"}  # Каналы
OFFSET_PER_FILE_SEC = 600        # 10 минут между файлами
AUDIO_BITRATE = "128k"           # Битрейт MP3
MAX_NO_SPEECH_PROB = 0.6
MIN_TEXT_LEN = 2
WHISPER_PROMPT = "Russian and Ukrainian speech. Court hearing or formal conversation. Do not translate. Transcribe verbatim."
```

**Алгоритм**:
```python
dialog = []
global_offset = 0.0

for video in sorted_video_files:
    print(f"Processing {video}")
    
    # Этап 1: Извлечение каналов
    for speaker, channel in SPEAKERS.items():  # S1:FL, S2:FR
        mp3 = f"extracted_audio/{video.stem}_{speaker}_{channel}.mp3"
        
        ffmpeg -i video -map 0:{STREAM_INDEX} -af "pan=mono|c0={channel}" \
               -ab {AUDIO_BITRATE} {mp3}
    
    # Этап 2: Транскрибация
    for speaker, channel in SPEAKERS.items():
        mp3 = f"extracted_audio/{video.stem}_{speaker}_{channel}.mp3"
        
        segments = transcribe_whisper(client, mp3)  # OpenAI API
        
        for segment in segments:
            dialog.append({
                "time": segment.start + global_offset,
                "end": segment.end + global_offset,
                "speaker": speaker,
                "text": segment.text
            })
    
    global_offset += OFFSET_PER_FILE_SEC
    time.sleep(1)  # Throttle API

# Этап 3: Сборка результата
dialog.sort(key=lambda x: x["time"])

# Сохранить в .txt
with open("dialog_timeline.txt", "w") as f:
    for row in dialog:
        f.write(f"[{format_ts_txt(row['time'])}] {row['speaker']}: {row['text']}\n")

# Сохранить в .srt
with open("dialog_timeline.srt", "w") as f:
    for idx, row in enumerate(dialog, 1):
        f.write(f"{idx}\n")
        f.write(f"{format_ts_srt(row['time'])} --> {format_ts_srt(row['end'])}\n")
        f.write(f"{row['speaker']}: {row['text']}\n\n")
```

**Функция-помощник `transcribe_whisper()`**:
```python
def transcribe_whisper(client: OpenAI, audio_file: Path) -> List[Dict]:
    with audio_file.open("rb") as f:
        response = client.audio.transcriptions.create(
            file=f,
            model="whisper-1",
            response_format="verbose_json",
            prompt=WHISPER_PROMPT,
            language=None,  # Auto-detect
        )
    
    data = response.model_dump()
    cleaned = []
    for seg in data.get("segments", []):
        text = seg.get("text", "").strip()
        if len(text) < MIN_TEXT_LEN or seg.get("no_speech_prob", 0) > MAX_NO_SPEECH_PROB:
            continue
        
        cleaned.append({
            "start": float(seg["start"]),
            "end": float(seg["end"]),
            "text": text,
        })
    
    return cleaned
```

**Статус**: ✅ **Полностью рабочий, используется в примере**
