# 📦 Release Notes — v0.1.0

**Дата релиза**: 28.03.2026  
**Тип релиза**: Native deployment (без Docker)

---

## Что вошло в релиз

### ✅ Phase 1
- Базовый pipeline реализован (`main.py`, `src/*`)
- Конфигурирование через YAML
- JSON-логирование обработки

### ✅ Phase 2 (DEMO)
- Стабилизация параметров ASR (`initial_prompt`, `beam_size`, `vad_filter`)
- Подтверждён DEMO прогон
- Зафиксирована стратегия каналов по успешному кейсу

### ✅ Phase 3 (PRODUCTION)
- Реализована chunked-обработка по времени
- Добавлены стадии `chunk_plan` и `chunk_XXX` в лог
- Корректный merge диалога с глобальными offset
- Пройдены smoke + extended tests

### ✅ Phase 4 (Deployment)
- Docker для релиза отклонён из-за ограничений целевого окружения
- Docker-наработки сохранены в ветке `phase4-docker-poc`
- Принят native путь: `.venv` + helper script + systemd unit
- Подтверждён запуск: `./scripts/run_v2t.sh config.production-smoke.yaml`
