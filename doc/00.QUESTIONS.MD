## 1️⃣ Выбор локального Whisper

### Вопрос 1.1: Какая модель Whisper?

**Контекст**: Прототип использует онлайн `whisper-1` API OpenAI. Для локального использования в ProxmoxVE контейнере нужно выбрать подходящую модель.

**Опции**:
1. **`openai-whisper`** (официальная Python библиотека)
   - Размер моделей: tiny (39M) → base (140M) → small (244M) → medium (769M) → large (2.9GB)
   - Скорость: медленно (~30-50 сек на минуту аудио для large)
   - Качество: высокое (large), но требует GPU

2. **`faster-whisper`** (оптимизированная версия от Systran)
   - Размер: те же, но с quantization (8-bit, 4-bit)
   - Скорость: быстро (~5-10 сек на минуту для medium)
   - Качество: близко к исходной, но меньше ОП

3. **`Ollama` + Whisper контейнер**
   - API-совместимый интерфейс
   - Легко масштабировать
   - Требует дополнительного контейнера

**Мой рецепт**: Рекомендую **`faster-whisper`** (лучший компромисс скорость/качество/память)

**Опции**:
1. **Прямая замена** (простейшая)
